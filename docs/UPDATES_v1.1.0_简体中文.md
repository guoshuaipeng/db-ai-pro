# 🎉 DB-AI v1.1.0 更新

## 三大核心功能升级

### 🚀 服务器端分页查询
- **智能保护**：自动添加 LIMIT 100，防止查询过多数据
- **内存优化**：每页仅加载 50 行，内存占用降低 95%
- **完整分页**：首页、上一页、下一页、末页，支持页码跳转
- **行数调整**：10-1000 行/页自由切换

### 🔍 智能缓存系统
- **快速启动**：从 5-10 秒 → 0.5 秒（提升 10 倍）
- **自动缓存**：展开过的数据库和表自动保存
- **无感刷新**：后台异步更新，用户无需等待

### 🗄️ 数据库管理
- **新建数据库**：支持字符集和排序规则选择
- **删除数据库**：三重安全确认，防误删
- **新建表**：快速跳转到 AI 建表页面

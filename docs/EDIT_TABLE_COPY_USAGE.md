# 编辑表 - 复制功能使用指南

## 🚀 快速开始

### 方法1：直接选择文本

最简单的方式，适用于复制表名、连接信息等。

**步骤**：
1. 在编辑表界面，鼠标移动到顶部的连接信息上
2. 光标会变成文本光标（I型）
3. 拖动鼠标选择需要的文本
4. 按 `Ctrl+C` 复制

**示例**：
```
选中: "test_connection - my_database.users"
复制: my_database.users
```

### 方法2：复制选中字段

适用于复制某几个特定字段的信息。

**步骤**：
1. 在表结构表格中选择一行或多行字段
   - 单击选择一行
   - 按住 `Ctrl` 点击选择多行
2. 右键点击选中的行
3. 选择"📋 复制选中字段"
4. 系统提示"已复制 X 个字段"

**复制结果**：
```
字段名	类型	可空	默认值	注释
--------------------------------------------------------------------------------
id	INT	NO	NULL	用户ID
name	VARCHAR(100)	NO	NULL	用户名
```

### 方法3：复制完整表结构

适用于导出完整的表结构文档。

**步骤**：
1. 在表结构表格中任意位置右键点击
2. 选择"📋 复制所有字段"
3. 系统提示"已复制完整表结构（X 个字段）"

**复制结果**：
```
表名: users
数据库: my_database
--------------------------------------------------------------------------------
字段名	类型	可空	默认值	注释
--------------------------------------------------------------------------------
id	INT	NO	NULL	用户ID
name	VARCHAR(100)	NO	NULL	用户名
email	VARCHAR(255)	YES	NULL	邮箱地址
created_at	DATETIME	NO	CURRENT_TIMESTAMP	创建时间
--------------------------------------------------------------------------------
主键: id
注释: 用户信息表
--------------------------------------------------------------------------------
索引列表:
PRIMARY KEY (id)
INDEX idx_email (email)
```

## 💡 实用技巧

### 技巧1：复制到Excel

**场景**：需要在Excel中整理字段清单

**操作**：
1. 右键点击表格 → "复制所有字段"
2. 打开Excel
3. 直接粘贴（`Ctrl+V`）
4. 数据会自动分列！

**效果**：
| 字段名 | 类型 | 可空 | 默认值 | 注释 |
|--------|------|------|--------|------|
| id | INT | NO | NULL | 用户ID |
| name | VARCHAR(100) | NO | NULL | 用户名 |

### 技巧2：快速复制表名

**场景**：只需要表名，不需要数据库前缀

**操作**：
1. 双击顶部信息中的表名部分
2. 表名会被自动选中
3. 按 `Ctrl+C` 复制

**技巧**：
- 双击选择一个单词
- 三击选择一整行

### 技巧3：复制字段名列表

**场景**：需要生成 `SELECT field1, field2, field3` 这样的字段列表

**操作**：
1. 复制选中字段或所有字段
2. 粘贴到文本编辑器
3. 使用查找替换：
   - 删除表头行
   - 删除类型、可空等列
   - 用逗号替换换行符

### 技巧4：对比两个表的差异

**场景**：需要查看两个表的字段差异

**操作**：
1. 打开表A的编辑界面，复制所有字段
2. 保存到文件 `table_a.txt`
3. 打开表B的编辑界面，复制所有字段
4. 保存到文件 `table_b.txt`
5. 使用对比工具（如Beyond Compare、WinMerge）对比

### 技巧5：生成表结构文档

**场景**：需要为项目生成数据库文档

**操作**：
1. 对每个表，使用"复制所有字段"
2. 粘贴到Word或Markdown文档
3. 添加标题和说明
4. 格式已经很清晰，无需额外调整

## 📋 复制格式说明

### Tab分隔符

复制的内容使用Tab（`\t`）作为列分隔符：
- 可以直接粘贴到Excel，自动分列
- 可以粘贴到文本编辑器，保持对齐
- 便于程序处理（如Python、Java等）

### 分隔线

使用连字符（`-`）创建分隔线：
- 美化显示效果
- 区分不同部分
- 提高可读性

### 完整信息

"复制所有字段"包含：
- ✅ 表名和数据库名
- ✅ 所有字段详细信息
- ✅ 主键信息
- ✅ 表注释
- ✅ 索引列表

## 🔍 故障排除

### 问题1：无法选择文本

**症状**：鼠标点击顶部信息，无法选择文本

**原因**：可能使用的是旧版本

**解决**：
- 确保更新到v1.2.0或更高版本
- 重启应用程序

### 问题2：右键菜单没有复制选项

**症状**：右键点击表格，没有"复制选中字段"选项

**原因**：没有选中任何行

**解决**：
- 先点击选中至少一行
- 然后右键点击

### 问题3：粘贴到Excel没有分列

**症状**：粘贴到Excel，所有内容在一列

**原因**：Excel设置问题

**解决**：
1. 粘贴后，选中这一列
2. 点击"数据" → "分列"
3. 选择"制表符"作为分隔符
4. 点击"完成"

### 问题4：复制的文本乱码

**症状**：粘贴后中文显示为乱码

**原因**：目标程序编码问题

**解决**：
- 先粘贴到记事本
- 保存为UTF-8编码
- 再从记事本复制到目标程序
